Document types
===================

Used in the :ref:`oai_dc <dctype>`, :ref:`qdc <dctermstype>` and :ref:`mets <metsstructmap>` formats, the document types are normalised as follow: 



OpenEdition Journals
-------------------------------

Possible values for OpenEdition Journals

* ``issue``
* ``part``
* ``article``
* ``review``
* ``bibliography``
* ``introduction``
* ``archaeological note``
* ``chapter``
* ``book``
* ``other``

OpenEdition Books
-------------------------------
Possible values for OpenEdition Books

* ``book``
* ``part``
* ``chapter``
* ``bibliography``
* ``appendix``
* ``index``
* ``foreword``
* ``afterword``
* ``bibliography``
* ``review``
* ``article``
* ``introduction``
* ``other``

Hypotheses
-------------------------------

Possible values for Hypotheses 

* ``post``

Calenda
-------------------------------

Possible values for Calenda 

* ``call for papers``
* ``colloquium``
* ``study days``
* ``seminar and workshop``
* ``miscellaneous information``
* ``scholarship, prize and job offer``
* ``lectures``
* ``summer school``
* ``call for tender``
